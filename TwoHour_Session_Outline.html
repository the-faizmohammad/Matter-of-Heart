2-Hour Live Session Outline

Zerodha-Style Authentication System in Node.js

Opening Hook (0–10 min)

  • Show real-world incidents: Instagram & Uber credential leaks
  • Explain why authentication & authorization is critical
  • Set up the project skeleton for Zerodha Secure Login Clone

Anchor Project: Zerodha Secure Login Clone

  • User registration with bcrypt hashing
  • JWT-based login and session protection
  • User and Admin role separation
  • Protected routes with middleware
  • Token expiry enforcement
  • Rate limiting to prevent brute-force attacks
  • Deployment to Render/Railway

Session Flow (120 minutes)

10–30 min — Password Security & Registration

  • Explain hashing, salting, and bcrypt cost factor
  • Implement /register route
  • Mini Exercise: Hash same password three times and compare outputs

30–55 min — JWT Authentication & Login

  • Create /login route issuing JWT
  • Show token decoding on jwt.io
  • Explain token expiry and refresh tokens

55–75 min — Role-Based Authorization

  • Create middleware to protect routes based on role
  • Implement Admin vs User dashboard
  • Mini Exercise: Add “Moderator” role with read-only access

75–90 min — Edge Case Demonstration

  • Break JWT payload → attempt privilege escalation
  • Remove expiry → show security risk
  • Simulate brute-force login → introduce rate limiting

90–105 min — Practical Challenges

  • Challenge 1: Email uniqueness check
  • Challenge 2: Protect a new /portfolio route
  • Challenge 3: Deploy live and share URL

105–115 min — Deployment & GitHub Setup

  • Configure environment variables
  • Deploy Node.js API to Render/Railway
  • Push project to GitHub with proper structure

115–120 min — Closing & Career Roadmap

  • Summarize authentication & authorization concepts
  • 90-day roadmap to 15–30 LPA backend roles
  • Key skills: JWT, OAuth, sessions, databases, caching, deployment

Edge Cases & Pitfalls Covered

  • Password stored in plaintext
  • JWT verification bypass
  • Privilege escalation via token tampering
  • Unlimited token lifetime
  • Client-side trust issues

Student Deliverables

  • Fully functional Zerodha-style authentication system
  • Live deployed application URL
  • GitHub repository with documentation
  • Reusable authentication boilerplate
  • Access to community support group

Tips to Ensure It Fits in 2 Hours

  • Pre-prepare the Project Structure: Node.js, Express, MongoDB, JWT library, bcrypt.

  • Efficient Screen Sharing: GitHub repo ready; learners type only critical parts (routes, middleware, JWT usage).
